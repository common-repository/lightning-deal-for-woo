Lightning Deal for WooCommerce is an extension of WooCommerce that allows site admin to create time-bound discount offers for the customers.

== Description ==

Lightning Deal for WooCommerce is a WooCommerce extension that empowers store owners to create time-limited discount offers, adding urgency and excitement to your sales strategy. This plugin seamlessly integrates with WooCommerce, allowing you to set up flash sales that will drive customer engagement and boost conversions.

= Free Version Features: =

* Create time-bound deals with customizable durations
* Display a claimed bar on shop and individual product pages
* Customize colors as per your brand
* Easy-to-use interface for managing deals
* Each deal can have multiple products. You can pick products individually or select a whole category.
* Discount type: Fixed amount discount only.

= Pro Version Features: =

* Everything in Free version
* Discount type: Fixed amount and percentage discounts.
* Claimed deals starting index: If you want to set an initial value of the claimed deal bar then you can use this feature.
* Max orders calculation method: Set deal claimed calculation method to Per Product or Per Deal
* Deal Statistics: See the total number of orders, total amount, and individual product sold for each deal.

Upgrade to the Pro version for advanced features and more flexibility in managing your lightning deals!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lightning-deal-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Lightning Deals to configure the plugin settings.
4. Go to Products > Lightning Deals and click 'Add New'. Set the deal details and products, then save.


== Frequently Asked Questions ==

= How do I create a new lightning deal? =

Go to Products > Lightning Deals and click 'Add New'. Set the deal details and products, then save.

= Can I set percentage discounts in the free version? =

No, percentage discounts are only available in the Pro version.

= I have some sales outside of the store, how can I reflect that in the deal? =

In the Pro version, we offer a feature called "Claimed deals starting index". This allows you to set an initial value for the claimed deal bar, reflecting sales that have occurred outside your online store. By setting this value, you can accurately represent the total number of items sold, including both online and offline sales.
